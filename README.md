java-vkmd
==============

A lighten music downloader for Vkontakte.

Requirements
------------

- Java >= 8.0

Run
---

```bash
$ ./vkmd
```

Developing
----------

- Gradle >= 4.3.1

### Build

```bash
$ ./gradlew build
```

### Run

```bash
$ ./gradlew run
```


### Testing

```bash
$ ./gradlew test
```
