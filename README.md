java-music-ytp
==============

?

Requirements
------------

- Java >= 8.0

Requirements for developing
---------------------------

- Gradle >= 4.3

Run
---

### For developing


```bash
$ ./gradlew run
```

### For using

```bash
$ java -jar musicytp.jar
```

Build
-----

```bash
$ ./gradlew build
```


Run tests
---------

```bash
$ ./gradlew test
```
