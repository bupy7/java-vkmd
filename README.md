java-vkmd
=========

[![Build Status](https://travis-ci.org/bupy7/java-vkmd.svg?branch=master)](https://travis-ci.org/bupy7/java-vkmd)

A lighten music downloader for Vkontakte.

![java-vkmd](screenshot.gif)

Requirements
------------

- Java >= 8.0

Run
---

```bash
$ ./vkmd
```

Developing
----------

- Gradle >= 4.3.1

### Build

```bash
$ ./gradlew build
```

### Run

```bash
$ ./gradlew run
```


### Testing

```bash
$ ./gradlew test
```
