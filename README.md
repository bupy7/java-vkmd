java-ytp-music
==============

A lighten desktop application to listen music from YouTube without video stream.

Requirements
------------

- Java >= 8.0

Run
---

```bash
$ ./ytp-music
```


Developing
----------

### Build

```bash
$ ./gradlew build
```

### Run

```bash
$ ./gradlew run
```


### Testing

```bash
$ ./gradlew test
```
